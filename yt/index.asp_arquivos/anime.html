ANIME
 

 

Com certeza você já assistiu um daqueles desenhos animados japoneses com personagens expressivos, olhos grandes e heróis super poderosos (alguns são cabeçudos também).
Esses desenhos são, para alguns de nós, algo sagrado, a razão de nossas vidas! (tá bem, exagerei, né?) Também, todos os dias quando passamos por bancas de jornais (outro local sagrado) somos bombardeados por toneladas de revistinhas que se lê de trás para frente...
"Trás para frente!?" Mas que diabos é isso tudo? Vamos perguntar ao site sabe tudo: O Wikipedia! Espero que as informações aqui contidas possam ser de grande ajuda para todos os que apreciam essa arte (e para os que não conhecem também) que vem dominando o Brasil e o mundo.

Lembre-se:
Nunca assistiu - Não julgue!
Se pensa que desenho japonês é só pancadaria... repense!
Se pensa que desenho japonês é só para crianças... repense novamente!
Assistiu e não gostou - Você está no site errado!
Você gosta - Então, boa diversão!

 

Anime (Anime por vezes escrito Animé ou Animê, sendo a real pronuncia Ánime, pois os japoneses assim o pronunciam) é o nome dado à animação japonesa. A palavra Anime tem significados diferentes para os japoneses e para os ocidentais. Para os japoneses, anime é tudo o que seja desenho animado, seja ele estrangeiro ou nacional. Para os ocidentais, anime é todo o desenho animado que venha do Japão.

Exemplos de animes conhecidos:
• Pokemón • Cavaleiros do Zodiaco • Dragon Ball • YuYu Hakusho • Digimon • Yu-Gi-Oh!
