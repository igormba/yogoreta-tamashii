Próximo episódio de Code Geass vaza antes na internet

 

Vazou hoje, no site Youtube, uma parte do terceiro episódio de Code Geass R2 que ainda não foi transmitido na TV do Japão. O vídeo continha os últimos 5 minutos do episódio e foi apagado rapidamente a pedido da Sunrise.

Um caso semelhante já aconteceu com o anime CODE-E, quando o vídeo de abertura vazou no Youtube antes da sua estréia, mas esta é a primeira vez que um episódio, mesmo apenas uma parte, vazou na internet antes de passar na TV.

Code Geass R2 passa todos os domingos às 17:00 horas na rede TBS. O último episódio foi o segundo e o terceiro está previsto para dia 20.

Update 16/04/2008
Uma nota oficial foi divulgada hoje informando que o episódio vazou acidentalmente de Bandai Channel, um site comercial de vídeo, durante uma manutenção do sistema. Code Geass continuará sendo exibido normalmente como programado.
